gcolor      gl_FragData[0] = color.rgb
gdepth      gl_FragData[1] = normal.xyz, terrain depth
gaux1       gl_FragData[4] = R0, smoothness, metalness
gaux2       gl_FragData[5] = torch light, ambient light, emission
gaux3       gl_FragData[6] = water depth

gnormal     gl_FragData[2] = lit + shadowed world
composite   gl_FragData[3] = first raytraced bounce

While this order may seen strange, I find I am unable to change the format of
the gaux textures, so only the first four attachments can be HDR buffers. I
would use a different data layout if I could.
